# 📊 نظام ERP الداخلي المتكامل

نظام إدارة موارد الشركة (ERP) متكامل يربط بين جميع أقسام الشركة ويوفر لوحة تحكم موحدة لإدارة العمليات.

## المميزات الرئيسية

### 1. قسم المشتريات
- إدارة الموردين والعلاقات معهم
- إدارة الأصناف والمنتجات
- مقارنة الأسعار من عدة موردين
- تتبع طلبات الشراء

### 2. قسم المخزن
- إدارة المخزون والكميات
- تحديد حد الأمان للأصناف
- تسجيل حركات المخزن (وارد/صادر)
- إشعارات تلقائية عند انخفاض الكمية
- تقارير المخزن

### 3. قسم التيليسيلز (CRM)
- إدارة بيانات العملاء
- تسجيل الطلبات
- تتبع نسبة نجاح الاستلام
- إدارة العلاقات مع العملاء

### 4. قسم المبيعات
- إنشاء وإدارة الفواتير
- ربط الفواتير بالطلبات والعملاء
- تتبع المبيعات والإيرادات
- إرسال الفواتير عبر الواتساب

### 5. قسم الشكاوى والدعم
- إنشاء وتتبع تذاكر الدعم الفني
- محادثات الواتساب المباشرة مع العملاء
- تصنيف الشكاوى حسب الأولوية
- تقارير الشكاوى والحلول

### 6. لوحة تحكم المدير
- رؤية شاملة لجميع العمليات
- تقارير وإحصائيات
- إدارة المستخدمين والصلاحيات
- إعدادات النظام

## التكنولوجيا المستخدمة

### Backend
- **Node.js + Express** - خادم الويب
- **tRPC** - نقل البيانات المكتوب بأمان
- **Drizzle ORM** - التعامل مع قاعدة البيانات

### Frontend
- **React 19** - مكتبة واجهات المستخدم
- **Tailwind CSS 4** - تصميم الواجهات
- **shadcn/ui** - مكونات واجهة جاهزة

### Database
- **MySQL/TiDB** - قاعدة البيانات الرئيسية

### Integration
- **WhatsApp Business API** - إرسال الرسائل والفواتير
- **Celery** - المهام غير المتزامنة
- **Redis** - الذاكرة المؤقتة

### Deployment
- **Docker & Docker Compose** - حاويات التطبيق
- **Nginx** - Reverse Proxy وموازن الحمل

## البدء السريع

### المتطلبات
- Docker و Docker Compose

# استنساخ المشروع

## دمج الواتساب

### الخطوات:

1. **إنشاء حساب WhatsApp Business**
   - انتقل إلى [Facebook Business Manager](https://business.facebook.com)
   - أنشئ تطبيق WhatsApp

2. **الحصول على بيانات الاتصال**
   - Business Phone Number ID
   - Access Token

3. **تحديث ملف .env**
   ```env
   WHATSAPP_BUSINESS_PHONE_ID=<your-id>
   WHATSAPP_ACCESS_TOKEN=<your-token>
   ```

4. **إعادة تشغيل الخدمات**
   ```bash
   docker-compose restart backend celery
   ```

## الأوامر المهمة

### تشغيل التطبيق
```bash
docker-compose up -d
```

### إيقاف التطبيق
```bash
docker-compose down
```

### عرض السجلات
```bash
docker-compose logs -f backend
```

### تشغيل أوامر Django
```bash
docker-compose exec backend python manage.py createsuperuser
```

### النسخ الاحتياطية
```bash
docker-compose exec postgres pg_dump -U user db_name > backup.sql
```
